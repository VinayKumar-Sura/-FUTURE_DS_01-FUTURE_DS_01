🎯 Task Objective
The objective of this task is to analyze business sales data to uncover:

Revenue trends over time
Top-performing products
High-value categories
Regional sales performance
The analysis aims to generate actionable business insights that support data-driven decision making.

🛠 Tools & Technologies Used
Python (Pandas, NumPy, Matplotlib, Seaborn)
Jupyter Notebook – for data cleaning, exploratory data analysis, and insights
Power BI – for interactive dashboard creation
GitHub – for version control and project documentation
📊 Analysis Overview
Cleaned and preprocessed raw sales data
Performed exploratory data analysis (EDA)
Identified revenue patterns, category-wise performance, and regional trends
Extracted meaningful insights relevant to business growth
All analytical steps and findings are documented in the Jupyter Notebook.

📈 Dashboard Overview
An interactive Power BI dashboard was created to visualize:

Total and regional sales performance
Category and product-level contribution
Key performance indicators (KPIs)
